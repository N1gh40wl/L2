Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		//defer close(ch) помог бы завершить программу верно
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
выведет все числа 0т 0 до 9 и упадет в дедлок из-за чтения открытого и пустого канала
